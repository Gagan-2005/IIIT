----------------------
"Feature Extraction:
- HuBERT Layer 3 embeddings (768-dim → PCA 128-dim)
- Mean-pooled over time for fixed-length representation

Classification:
- Random Forest with 300 estimators
- SMOTE for class balancing
- 80/20 train/test split (speaker-independent)"

------------------
"The final model achieves 99.73% accuracy on the test set. Testing on

- Karnataka: 99.2% precision
- Kerala: 100% precision
- Tamil Nadu: 99.7% precision"

SECTION 4: LIMITATIONS
----------------------
"While the model performs excellently on test data, we identified key 
limitations when testing beyond the training distribution:

1. NEW SPEAKER GENERALIZATION:
   - Confidence drops to 30-50% for completely unseen speakers
   - Root cause: Training data from specific recording sessions
   - Model learned speaker-specific rather than pure accent patterns

2. AGE GAP:
   - Child speaker performance significantly lower (20-40% confidence)
   - Different voice characteristics and limited training data

3. SYNTHETIC AUDIO:
   - TTS audio correctly identified as 'Unknown' (positive finding)"

SECTION 5: FUTURE WORK
-----------------------
"To address the generalization limitation, we propose data augmentation:

PROPOSED SOLUTION (Code provided in documentation):
- Pitch shifting (±2 semitones) to simulate voice variation
- Time stretching (0.9x-1.1x) for speaking rate diversity
- Additive noise for recording condition robustness

Expected Benefits:
- 6x dataset increase (1 original + 5 augmented per sample)
- Force model to learn accent-specific rather than speaker-specific features
- Improve new speaker performance

Additional Enhancements:
- Speaker normalization techniques (i-vectors, x-vectors)
- Expanded data collection with diverse recording conditions
- Ensemble methods combining HuBERT and MFCC features

Note: Augmentation code is documented but not yet implemented, as it 
would require significant retraining time."

SECTION 6: CONCLUSION
----------------------
"This project successfully demonstrates accent recognition from speech 
using self-supervised learning (HuBERT) with 99.73% accuracy on the test 
set. The model works exceptionally well for speakers in the training 
distribution but shows expected limitations on completely new speakers. 
We've identified this generalization challenge and proposed a concrete 
solution (data augmentation) backed by implementation-ready code. The 
project includes a working web application and comprehensive documentation."
